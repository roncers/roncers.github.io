# first-p5

A generative art sketch built with [p5.js](https://p5js.org). Particles start at the center of the screen and drift randomly, gradually shifting color over time.

## How it works

- A full-window canvas is created on load.
- A set of particles spawn at the center, their count based on the window size.
- Each frame, every particle moves by a small random offset (random walk).
- The fill color evolves randomly across RGB channels, producing smooth color transitions.

## Getting started

Simply open `index.html` in a browser — no build step required.
